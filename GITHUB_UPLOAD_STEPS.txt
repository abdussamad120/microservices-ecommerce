# How to Upload Your Project to GitHub

Follow these steps exactly to upload your `Submission_Package/02_Source_Code` to GitHub.

## Prerequisite: Create a Repository on GitHub
1.  Go to [github.com/new](https://github.com/new).
2.  **Repository Name:** `microservices-ecommerce` (or a name of your choice).
3.  **Description:** "A scalable microservices e-commerce platform built with Next.js, Express, and Kafka."
4.  **Visibility:** Public (so recruiters can see it).
5.  **Initialize:** Do NOT check "Add a README", ".gitignore", or "License". We already have these.
6.  Click **Create repository**.

## Step 1: Initialize Git (Locally)
Running this command inside your terminal in this folder:

```bash
cd "c:\Users\abdus\ABDUSSAMADRAEEN\microservices-ecommerce-main\Submission_Package\02_Source_Code"
git init
```

## Step 2: Add Files
Add all the files to the staging area. Our `.gitignore` will ensure we don't upload junk.

```bash
git add .
```

## Step 3: Commit
Save your changes with a professional message.

```bash
git commit -m "Initial commit: Microservices E-commerce Platform V1.0"
```

## Step 4: Link to GitHub
Replace `<YOUR_GITHUB_URL>` with the URL of the repository you just created (e.g., `https://github.com/abdussamad/microservices-ecommerce.git`).

```bash
git branch -M main
git remote add origin <YOUR_GITHUB_URL>
git push -u origin main
```

## Done! 🚀
Your code is now live on GitHub.

> **Note on Xiaomi Project:**
