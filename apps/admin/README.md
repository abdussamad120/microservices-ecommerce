# E-commerce Admin Dashboard

The administrative panel for managing the E-commerce platform.

## Overview

This application allows store administrators to:
- Manage the Product Catalog (Add, Edit, Delete products)
- Monitor and update Inventory
- View and manage Customer Orders
- Visualize sales data and analytics

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **UI Components**: Shadcn UI / Radix UI
- **Styling**: TailwindCSS
- **Charts**: Recharts
- **Forms**: React Hook Form + Zod

## Running Locally

To run only the admin application:

```bash
pnpm dev
```

The application will start at [http://localhost:3501](http://localhost:3501).
